# Spherid / Drift Game Concept: Concept Art Prompt Packet

**Status:** Planning packet  
**Date:** 2026-04-24  
**Purpose:** Provide prompt-ready concept art directions for the world, biomes, urchin character, shape presences, vehicles, and revelation states.

---

## 1. Prompting Rules

All prompts should stay aligned with the project’s visual north star:

- luminous painterly-sculptural style,
- stylized rather than realistic,
- emotionally sincere,
- strong material readability,
- silent pilgrimage tone,
- rare sacred-surreal intensification.

Avoid in prompts:
- photorealism,
- gritty realism,
- generic cartoon look,
- overdesigned sci-fi machinery,
- hyper-busy fantasy clutter,
- horror tone unless explicitly desired.

Useful style phrases:
- painterly sculptural landscape
- luminous stylized game art
- elegant stylized materials
- atmospheric cinematic composition
- sacred-surreal but readable
- tactile stylized environment design

## 10. Prompt Refinement Notes

When refining generated images, tune for:
- clearer silhouette,
- less realism,
- fewer small details,
- stronger environmental shape language,
- stronger atmosphere,
- more readable traversal surfaces,
- more sacred and sincere tone,
- less generic fantasy clutter.

If an image feels:
- too sci-fi,
- too realistic,
- too busy,
- too cartoony,
- too dark or grim,

pull it back toward:
- painterly,
- sculptural,
- luminous,
- readable,
- emotionally quiet,
- symbolically intentional.

# Spherid / Drift Game Concept: Image Generation Batch

**Status:** Planning batch  
**Date:** 2026-04-24  
**Purpose:** Provide a structured batch of image-generation targets that can be used to quickly explore the game’s world, character, symbolic language, and movement fantasy.

---

## 1. Batch Strategy

Do not generate random pretty scenes.

The image batch should answer concrete questions:
- What is the visual north star?
- What does Drift look like in-world?
- What do the first three biomes feel like?
- What do shape presences look like?
- What do vehicles/devices look like?
- What does a revelation state look like without breaking style coherence?

The batch should start broad, then narrow.

Recommended order:

1. global style lock
2. character look
3. early biome anchors
4. world symbolism and shape presences
5. devices and vehicles
6. revelation-state tests

## 9. Recommended Batch Order

Run the batches in this order:

1. Batch A
2. Batch B
3. Batch C
4. Batch G
5. Batch D
6. Batch E
7. Batch F

That order helps lock:
- overall style,
- main character,
- early biome look,
- symbolic/material language,
before spending effort on rarer or more specialized images.

---

## 10. Evaluation Criteria

Keep images that show:
- strong silhouette,
- painterly-sculptural terrain,
- readable traversal routes,
- emotionally sincere tone,
- material distinction without realism,
- coherent relation between character and world,
- sacred-surreal restraint.

Reject images that feel:
- too photoreal,
- too generic fantasy,
- too noisy,
- too sci-fi mechanical,
- too cartoonish,
- too dark or grim,
- too psychedelic in baseline scenes.

---

## 11. Final Recommendation

Use this batch to lock a visual identity before going too deep into asset-specific thinking.

The key visual promise to preserve across all generated images is:

**A tiny expressive rolling pilgrim moving through a beautiful world where shape, light, resonance, and landscape all belong to the same spiritual material language.**

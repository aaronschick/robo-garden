# Urchin Game Concept: Biome Art Bible

**Status:** Planning bible  
**Date:** 2026-04-24  
**Purpose:** Define the visual identity, material language, lighting mood, and production priorities for each core biome in the urchin game concept.

---

## 1. Usage

This document should guide:
- concept art,
- material direction,
- lighting targets,
- environmental storytelling,
- set dressing priorities,
- FX planning,
- color script continuity.

Each biome should feel visually distinct while still clearly belonging to the same world.

Shared global rules:
- sculptural terrain,
- painterly stylization,
- physically readable materials,
- strong atmospheric depth,
- restrained but meaningful surrealism,
- traversal readability always preserved.

---

## 2. Moss Cathedral

### Emotional Role

Awakening, tenderness, shelter, early vulnerability.

### Visual Keywords

- soft
- damp
- protective
- oversized
- breathing
- rooted

### Shape Language

- rounded hollows
- thick root arches
- soft basins
- layered root curtains
- nested cavities
- gentle slopes

### Material Direction

- velvety moss
- dew membranes
- damp bark-like surfaces
- soft soil
- translucent spores

### Lighting

- filtered canopy glow
- cool-green shadows
- warm dew highlights
- thin volumetric shafts
- intimate local contrast

### Color Direction

Primary:
- moss green
- muted blue-green
- dark bark umber

Accents:
- dew gold
- pale pollen cream
- gentle cyan luminescence

### Set Dressing

- seed pods
- folded leaf-like membranes
- moss bowls
- hanging tendrils
- spore clusters
- root ladders

### FX Priorities

- drifting pollen
- dew sparkle
- soft impact puffs
- low mist pockets
- subtle plant response to pulse

### Readability Notes

Keep early traversal paths clear through:
- brighter route moss,
- softer curvature,
- framing roots,
- distinct shelter pockets.

### Production Note

This biome can look rich with:
- one strong moss shader,
- a dew shader,
- layered fog,
- a modest prop kit.

## 10. Cross-Biome Consistency Rules

To keep all biomes in the same game:

- Maintain the same overall painterly-sculptural base.
- Use recurring resonance motifs.
- Keep material response believable within each biome’s stylized logic.
- Let the urchin remain legible in all palettes.
- Ensure each biome supports its movement fantasy visually.
- Reserve the strongest surreal intensification for revelation moments only.

---

## 11. Final Recommendation

The biome art direction should be built around contrast in:
- material personality,
- mood,
- movement rhythm,
- lighting structure,
- color script,
- environmental motion.

Not around realism level.

The player should be able to recognize each biome instantly by:
- silhouette,
- palette,
- atmosphere,
- and the feeling of moving through it.

# Spherid / Drift Naming Canon

**Status:** Naming canon  
**Date:** 2026-04-24  
**Purpose:** Formalize the naming system for the main character and related world forms so future planning and implementation use consistent language.

---

## 1. Canon Decision

The old codename `Urchin` no longer matches the project’s direction.

The new canon is:

- **Species/Form:** `Spherid`
- **Individual/Title:** `Drift`

Use this as the default in future sessions.

---

## 2. Meaning

### Spherid

`Spherid` is the formal shape-world term.

It should read as:
- geometric,
- species-like,
- world-native,
- comparable to names like `oloid` and `tricylinder`.

It is the correct equivalent when referring to the being as a type of form in the world.

### Drift

`Drift` is the individual or symbolic title.

It should read as:
- quiet,
- lyrical,
- motion-defined,
- slightly fateful,
- open to multiple meanings.

It works because it implies:
- passage,
- being carried,
- uncertainty early,
- grace later,
- movement as identity.

---

## 3. Usage Rules

### Use `Spherid` when referring to:

- the species or form class,
- cosmology,
- shape philosophy,
- worldbuilding taxonomy,
- “what kind of being this is.”

Examples:
- “The pilgrim is a spherid.”
- “Spherids belong most directly to the Circular family.”
- “A spherid learns the world through contact.”

### Use `Drift` when referring to:

- the main character as an individual,
- emotional or story beats,
- chapter invocations,
- scene-specific writing,
- symbolic role.

Examples:
- “Its name is Drift.”
- “Drift learned that motion could be written.”
- “Drift crossed into sunlight.”

### Use both together when clarity helps:

- “Drift is a spherid.”
- “The player controls Drift, a spherid pilgrim.”

---

## 4. Related Form Names

This naming canon implies a broader system where:

- the world uses formal geometric form names,
- individuals may or may not have symbolic titles,
- not every being needs a personal name.

Examples:
- `spherid`
- `oloid`
- `tricylinder`

Potential symbolic titles:
- `Drift`
- `Hush`
- `Verge`
- `Lift`
- `Trace`

This should help the world feel coherent and non-generic.

---

## 5. Invocation Voice

These lines define the intended voice:

- “The pilgrim is a spherid.”
- “Its name is Drift.”
- “The oloid crossed the dunes.”
- “A tricylinder waited at the threshold.”

This voice should be:
- rare,
- ceremonial,
- quiet,
- spare,
- world-defining.

It is not general prose for every scene. It is chapter or threshold language.

---

## 6. Transition Rule For Existing Docs

When updating older planning docs:

- replace `urchin` with `spherid` when discussing species/form, body, cosmology, or systemic design,
- replace `urchin` with `Drift` when discussing the protagonist as an individual in story language,
- use `Drift, a spherid` when a sentence needs both identity and taxonomy.

This is a tone-sensitive replacement, not a blind global search-and-replace.

---

## 7. Recommendation

Future planning should assume:

- the playable protagonist is **Drift**,
- Drift’s form/species is **spherid**,
- `Urchin` is now historical dev language only and should be phased out of forward-looking concept work.

# Urchin Game Concept: Symbol To Biome Mapping

**Status:** Planning map  
**Date:** 2026-04-24  
**Purpose:** Map shape families, glyph families, and symbolic meanings across the game’s biome sequence.

---

## 1. Purpose

This document helps ensure:
- symbols recur coherently,
- each biome has a dominant visual-philosophical identity,
- movement mechanics and worldbuilding stay aligned,
- revelation states can synthesize prior motifs meaningfully.

---

## 2. Mapping Rules

Each biome should emphasize:
- 1 dominant shape philosophy,
- 1 secondary shape philosophy,
- 2-3 major glyph families,
- a movement meaning,
- a symbolic emotional meaning.

The biome should not try to represent all possible motifs equally.

## 12. Recommendation

Use this mapping as a constraint for:
- concept art,
- glyph placement,
- level motifs,
- puzzle design,
- revelation-state synthesis.

If a symbol appears often in a biome, it should be there for a reason.

The player should gradually feel that:

**every place in the world has its own movement philosophy.**
